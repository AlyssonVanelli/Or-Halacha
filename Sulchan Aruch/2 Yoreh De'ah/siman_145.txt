SIMAN 145

1. Qualquer coisa que não tenha sido feita por mãos humanas e não tenha sido criada por uma pessoa, mesmo que tenha sido adorada, é permitida para uso. Portanto, se alguém se prostrar diante de uma montanha, ela não se torna proibida. Mesmo as pedras da montanha que se soltaram e ainda estão no lugar, se forem adoradas ali, não se tornam proibidas. O mesmo se aplica a rios, fontes públicas e árvores que não foram plantadas para idolatria; eles não se tornam proibidos. No entanto, embora o corpo da árvore não se torne proibido quando alguém se prostra diante dela, todos os ramos, folhas e frutos que ela produzir enquanto for adorada são proibidos para uso.

2. Algo que não foi feito por mãos humanas e foi adorado, mesmo que o objeto em si seja permitido para uso, seu revestimento é proibido para uso. (Os acessórios da montanha são como seu revestimento e são proibidos, mas as oferendas à montanha não são proibidas).

3. Uma casa que foi construída desde o início para ser adorada, ou se alguém se prostrar diante de uma casa já construída, ela é proibida para uso. Se a casa já estava construída e foi rebocada ou pintada para idolatria, até que algo novo tenha sido adicionado, deve-se remover o que foi adicionado, e o novo é proibido para uso porque foi feito para adoração, mas o restante da casa é permitido. Se um ídolo foi colocado dentro da casa, enquanto ele estiver lá, a casa é proibida para uso. Se for removido, a casa é permitida. (Isso é apenas se a casa não foi designada para isso, mas se foi, ela só é permitida se o ídolo for removido de uma maneira que indique anulação, ou seja, não colocá-lo lá novamente).

4. Uma pedra que foi esculpida para adoração é proibida para uso. Se já estava esculpida e foi pintada ou decorada para ser adorada, mesmo que a pintura ou decoração tenha sido feita no corpo da pedra, deve-se remover o que foi adicionado, e isso é proibido para uso, mas o restante da pedra é permitido. (Em idolatria, onde a anulação é eficaz, mesmo que nada tenha sido removido, apenas o que foi pintado é permitido se for feito de maneira a indicar anulação).

5. Uma pedra sobre a qual foi colocado um ídolo de idolatria é proibida enquanto o ídolo estiver sobre ela. Se for removido, a pedra é permitida.

6. Uma árvore que foi plantada desde o início para ser adorada é proibida para uso. Se a árvore já estava plantada e foi cortada e esculpida para idolatria, mesmo que tenha sido enxertada ou tenha brotado ramos no corpo da árvore, deve-se cortar os ramos, e eles são proibidos para uso, mas o restante da árvore é permitido. Uma árvore sob a qual ídolos são colocados é proibida para uso enquanto eles estiverem lá. Se forem removidos, a árvore é permitida, pois a árvore em si não é adorada.

7. Embora o solo do mundo não se torne proibido, se alguém fizer uma ação nele, como cavar poços, valas e cavernas para idolatria, ele se torna proibido.

8. Animais vivos não se tornam proibidos, pois mesmo que alguém se prostrar diante de seu próprio animal, ele não se torna proibido. Se uma ação for feita, como abatê-lo para idolatria, mesmo que apenas um sinal tenha sido cortado, ele se torna proibido, mesmo que não pertença à pessoa. No entanto, apenas um idólatra pode proibir algo que não é dele, mas um judeu não pode proibir o que pertence a outro, pois certamente ele não tem a intenção de fazer mais do que causar sofrimento. Alguns dizem que mesmo se ele tiver uma parte nisso, ele não proíbe. Se um judeu for um apóstata ou se ele foi advertido e aceitou a advertência, ele proíbe. (Veja acima, Siman 4. Oferendas de animais e seus acessórios são proibidos).

9. Embora animais vivos não se tornem proibidos, se forem trocados por ídolos, eles se tornam proibidos. Mas trocas de trocas, como trocar animais por trocas de ídolos, são permitidas. Alguns proíbem isso também.


--- EXPLICAÇÃO PRÁTICA ---

Este texto do Shulchan Aruch aborda leis relacionadas à idolatria, um conceito importante no judaísmo que se refere à adoração de ídolos ou objetos como deuses. Vamos explicar cada seção de forma prática:

1. **Objetos Naturais e Idolatria**: Coisas que não foram feitas por humanos, como montanhas, rios e árvores, não se tornam proibidas se forem adoradas. Isso significa que, mesmo que alguém se curve diante de uma montanha, ela ainda pode ser usada. No entanto, se uma árvore for adorada, seus frutos e folhas produzidos durante esse tempo são proibidos.

2. **Revestimentos de Objetos Naturais**: Embora o objeto natural em si não se torne proibido, qualquer revestimento ou acessório adicionado a ele para fins de adoração é proibido. Por exemplo, se uma montanha tiver decorações feitas para idolatria, essas decorações não podem ser usadas.

3. **Casas e Idolatria**: Se uma casa é construída especificamente para adoração, ela é proibida. Se uma casa já existente for decorada para idolatria, as decorações devem ser removidas. Se um ídolo for colocado dentro de uma casa, a casa é proibida enquanto o ídolo estiver lá, a menos que o ídolo seja removido de forma que indique que não será mais usado para adoração.

4. **Pedras e Idolatria**: Uma pedra esculpida para adoração é proibida. Se uma pedra já esculpida for pintada para idolatria, a pintura deve ser removida. A pedra em si não é proibida, apenas as adições feitas para idolatria.

5. **Pedras com Ídolos**: Se um ídolo é colocado sobre uma pedra, a pedra é proibida enquanto o ídolo estiver lá. Removendo o ídolo, a pedra volta a ser permitida.

6. **Árvores e Idolatria**: Árvores plantadas para adoração são proibidas. Se uma árvore já existente for modificada para idolatria, as partes modificadas devem ser removidas. Se ídolos forem colocados sob uma árvore, ela é proibida enquanto os ídolos estiverem lá.

7. **Solo e Idolatria**: O solo em si não se torna proibido, mas se alguém fizer algo nele, como cavar para fins de idolatria, essa parte do solo se torna proibida.

8. **Animais e Idolatria**: Animais vivos não se tornam proibidos se forem adorados. No entanto, se um animal for sacrificado para idolatria, ele se torna proibido. Apenas um idólatra pode tornar algo que não é dele proibido, mas um judeu não pode fazer isso com propriedade de outro.

9. **Trocas de Animais por Ídolos**: Animais trocados diretamente por ídolos se tornam proibidos. No entanto, se um animal for trocado por algo que foi trocado por um ídolo, ele é permitido, embora haja opiniões que proíbam isso também.

Essas leis refletem a preocupação do judaísmo em evitar qualquer associação com práticas idólatras, mantendo a pureza da adoração a Deus. Na vida contemporânea, essas regras ajudam a orientar os judeus sobre como lidar com objetos e práticas que possam ter conotações idólatras.
