SIMAN 143

1. É proibido construir com o idólatra uma cúpula onde se colocam ídolos. E se transgrediu e construiu, mesmo que seja para a própria idolatria, seu pagamento é permitido. No entanto, é permitido construir desde o início o salão ou o pátio que contém essa cúpula.

2. Se a idolatria possui um jardim ou um banho público, e o benefício que deles advém é para os sacerdotes, é permitido usufruir deles sem benefício monetário. Mas é proibido usufruir deles com benefício monetário, mesmo que outros compartilhem o benefício com os sacerdotes. E se o benefício que advém deles não é para os sacerdotes, mas sim para seus adoradores, é permitido usufruir deles mesmo com benefício monetário, mesmo que o jardim e o banho público pertençam exclusivamente à idolatria. Nota: Alguns dizem que é proibido usufruir se o benefício vai para os sacerdotes apenas quando estão no pátio da própria idolatria, mas quando não estão diante dela, mesmo que o benefício vá para os sacerdotes, é permitido usufruir, a menos que o benefício vá para a própria idolatria. E pode-se confiar nisso para ser leniente.

3. Flautas de idolatria são proibidas para lamentações.

4. Lojas de idolatria são proibidas de serem alugadas. Nota: Se o aluguel vai para o estado e eles compram suas necessidades, é permitido.

5. Coletadores que cobram impostos para a idolatria é proibido dar-lhes. Nota: Aqui também, não é proibido a menos que o pagamento vá para as necessidades da própria idolatria, mas se vai para o tesouro do estado, mesmo que comprem necessidades para a idolatria, é permitido.


--- EXPLICAÇÃO PRÁTICA ---

Vamos explicar o Siman 143 do Shulchan Aruch Yoreh De'ah de forma prática e acessível:

1. **Construção de Estruturas para Idolatria**: 
   - **Proibição**: É proibido ajudar a construir uma cúpula ou estrutura que será usada para colocar ídolos. Isso se refere a não participar na criação de locais de culto para práticas idólatras.
   - **Exceção**: Se alguém já construiu uma estrutura dessas, o pagamento recebido por esse trabalho é permitido, mesmo que a construção tenha sido para idolatria.
   - **Construções Permitidas**: É permitido construir outras partes do edifício, como salões ou pátios, que não são diretamente para os ídolos.

2. **Uso de Jardins e Banhos Públicos de Idolatria**:
   - **Sem Benefício Monetário**: Se um jardim ou banho público pertencente a uma prática idólatra beneficia apenas os sacerdotes dessa religião, você pode usá-los, mas sem obter qualquer ganho financeiro.
   - **Com Benefício Monetário**: Se o uso desses locais gera lucro, é proibido se o lucro vai para os sacerdotes. No entanto, se o lucro vai para os adoradores em geral, é permitido.
   - **Nota Adicional**: Alguns dizem que é proibido usufruir desses locais se o benefício vai para os sacerdotes apenas quando estão no local de idolatria. Fora desse contexto, é permitido, a menos que o lucro vá diretamente para a idolatria.

3. **Instrumentos Musicais de Idolatria**:
   - **Proibição**: Instrumentos musicais usados em práticas idólatras, como flautas, não devem ser usados em cerimônias de luto judaicas.

4. **Aluguel de Lojas de Idolatria**:
   - **Proibição**: Não se deve alugar lojas que pertencem a práticas idólatras.
   - **Exceção**: Se o aluguel vai para o estado e não diretamente para a idolatria, é permitido.

5. **Pagamento de Impostos para Idolatria**:
   - **Proibição**: Não se deve pagar impostos que vão diretamente para sustentar práticas idólatras.
   - **Exceção**: Se os impostos vão para o tesouro do estado e não diretamente para a idolatria, é permitido, mesmo que parte do dinheiro seja usado para necessidades idólatras.

Essas leis refletem a preocupação em evitar qualquer apoio direto a práticas idólatras, mantendo a integridade da prática judaica. Na vida contemporânea, essas diretrizes ajudam a orientar como os judeus devem interagir com instituições ou práticas que não estão alinhadas com os princípios do monoteísmo judaico.
