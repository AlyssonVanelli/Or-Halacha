SIMAN 322

**Lei sobre o que é "nascido" no Shabat. Contém seis seções:**

1. Um ovo que foi posto no Shabat é proibido até mesmo de ser movido. Mesmo que se misture com mil outros, todos são proibidos. É permitido cobri-lo com um utensílio para que não quebre, desde que não se toque nele.

2. Se o Shabat ocorre após um Yom Tov (feriado judaico) ou antes dele, um ovo posto em um desses dias é proibido no outro.

3. Frutas que caíram de uma árvore no Shabat são proibidas de serem consumidas naquele dia, mas são permitidas imediatamente após o anoitecer.

4. Alimentos para animais não são considerados como "conserto de utensílio", portanto, é permitido cortar palha ou feno com uma faca para limpar os dentes. No entanto, um graveto que não é alimento para animais é proibido de ser usado para limpar os dentes.

5. É permitido mover madeiras aromáticas para cheirá-las ou para abaná-las sobre um doente. Pode-se esfregá-las para cheirar, tanto as duras quanto as macias.

6. Quem distribui porções para os membros da casa no Shabat pode lançar sortes para determinar a quem cada porção pertence, desde que as porções sejam iguais, para evitar ciúmes. No entanto, com outras pessoas, é proibido, pois pode levar a disputas sobre medidas e pesos. Distribuir porções desiguais e lançar sortes sobre elas é proibido, mesmo entre os membros da casa e durante a semana, devido ao jogo de azar. Alguns dizem que com filhos e membros da casa é permitido lançar sortes mesmo sobre porções desiguais, pois não se importam. Nota: É proibido lançar sortes no Shabat, mesmo através de um não-judeu [Maharil].


--- EXPLICAÇÃO PRÁTICA ---

O Siman 322 do Shulchan Aruch Orach Chayim aborda várias leis relacionadas ao Shabat, um dia sagrado de descanso no judaísmo. Vamos explicar cada seção de forma prática:

1. **Ovos postos no Shabat**: Se uma galinha põe um ovo no Shabat, esse ovo não pode ser usado ou movido durante o dia. Isso se deve a uma categoria de proibições chamada "muktze", que se refere a objetos que não estavam preparados para uso antes do Shabat. Mesmo que esse ovo se misture com muitos outros, todos se tornam proibidos de serem usados no Shabat. No entanto, é permitido proteger o ovo para que não quebre, desde que não se toque nele diretamente.

2. **Shabat e Yom Tov consecutivos**: Se o Shabat cai imediatamente antes ou depois de um Yom Tov (feriado judaico), um ovo posto em qualquer um desses dias não pode ser usado no outro dia. Isso se deve à continuidade das restrições de trabalho entre esses dias sagrados.

3. **Frutas caídas de árvores**: Se frutas caem de uma árvore no Shabat, não podem ser consumidas até o término do Shabat, após o anoitecer. Isso também está relacionado ao conceito de "muktze", pois as frutas não estavam disponíveis para uso antes do Shabat.

4. **Alimentos para animais e limpeza dental**: É permitido cortar alimentos para animais, como palha ou feno, para limpar os dentes, pois isso não é considerado "conserto de utensílio", uma atividade proibida no Shabat. No entanto, usar gravetos que não são alimentos para animais é proibido para esse fim.

5. **Madeiras aromáticas**: É permitido mover madeiras aromáticas para cheirá-las ou usá-las para abanar um doente, pois isso não é considerado trabalho. Pode-se também esfregar essas madeiras para liberar seu aroma, independentemente de serem duras ou macias.

6. **Distribuição de porções e sorteios**: No Shabat, ao distribuir porções de comida entre os membros da casa, é permitido lançar sortes para decidir quem recebe o quê, desde que as porções sejam iguais, para evitar ciúmes. Com pessoas de fora, isso é proibido, pois pode levar a disputas. Lançar sortes sobre porções desiguais é proibido, mesmo durante a semana, por ser considerado jogo de azar. No entanto, alguns permitem lançar sortes entre membros da casa, mesmo com porções desiguais, pois não se importam. É importante notar que lançar sortes no Shabat é proibido, mesmo que feito por um não-judeu.

Essas leis refletem a importância de manter o Shabat como um dia de descanso e espiritualidade, evitando atividades que possam ser consideradas trabalho ou que causem disputas e preocupações.
