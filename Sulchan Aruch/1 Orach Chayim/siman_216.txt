SIMAN 216

**Leis sobre a bênção do aroma. Contém 14 seções:**

1. É proibido desfrutar de um bom aroma até que se faça a bênção antes de cheirá-lo. No entanto, após cheirá-lo, não é necessário fazer a bênção.

2. Se a fonte do aroma é uma árvore ou um tipo de árvore, a bênção é "Borê Atzei Besamim" (Criador das árvores aromáticas). Se for uma erva, a bênção é "Borê Isvei Besamim" (Criador das ervas aromáticas). Se não for nem árvore nem erva, como o almíscar, a bênção é "Borê Minei Besamim" (Criador de tipos de aromas). Se o aroma provém de um fruto adequado para consumo, a bênção é "Hanoten Reiach Tov Baperot" (Que dá um bom aroma aos frutos), mas isso se aplica quando se pega o fruto para cheirá-lo ou para comê-lo e cheirá-lo. Se foi pego apenas para comer e não se teve a intenção de cheirá-lo, mesmo que exale um bom aroma, não se faz a bênção. Em todos os casos, se a bênção "Borê Minei Besamim" foi dita, cumpre-se a obrigação. Portanto, em caso de dúvida, a bênção "Borê Minei Besamim" deve ser feita. Para noz-moscada, canela, cravo e outros aromas comestíveis, a bênção é "Hanoten Reiach Tov Baperot".

3. Para a rosa (e para a canela, que é a casca da árvore), água de rosas, incenso e similares, a bênção é "Borê Atzei Besamim".

4. Para o óleo de bálsamo, a bênção é "Borê Shemen Arev" (Criador do óleo agradável).

5. Para o azeite de oliva que foi esmagado ou moído até que seu aroma se tornasse perceptível, a bênção é "Borê Atzei Besamim".

6. Para um óleo perfumado, como o óleo da unção, se for feito de árvores aromáticas, a bênção é "Borê Atzei Besamim"; se for de ervas aromáticas, "Borê Isvei Besamim". Se contiver tanto árvores quanto ervas, a bênção é "Borê Minei Besamim". Se o óleo foi filtrado e os aromas removidos, alguns dizem que a bênção é "Borê Shemen Arev", enquanto outros dizem que não se faz nenhuma bênção, pois é um aroma sem substância. Dado que há dúvida, é aconselhável evitar cheirá-lo.

7. Para "Simlak" e "Chilfi Dima", a bênção é "Borê Atzei Besamim". "Simlak" é interpretado por alguns como alecrim e por outros como jasmim. Outros ainda dizem que é uma erva com três fileiras de folhas, cada uma com três folhas. "Chilfi Dima" é o espicanardo, conhecido como "Ishpik".

8. Para "Sigli", que é a violeta, a bênção é "Borê Isvei Besamim".

9. Para "Nargis", que é o narciso, se cresce no jardim, a bênção é "Borê Atzei Besamim"; se cresce no campo, "Borê Isvei Besamim".

10. Se há diante de alguém árvores aromáticas, ervas aromáticas e tipos de aromas, faz-se a bênção apropriada para cada um. Se a bênção "Borê Minei Besamim" foi feita para todos, cumpre-se a obrigação. Se é necessário dar prioridade à bênção das árvores sobre as ervas, veja acima no Siman 211.

11. Se trouxerem diante de alguém mirto e óleo para cheirar, se as bênçãos forem iguais, faz-se a bênção sobre o mirto e isenta-se o óleo. Se não forem iguais, faz-se a bênção sobre o mirto primeiro.

12. Faz-se a bênção sobre o "Mugmar" (aromas colocados sobre brasas para cheirar) quando a fumaça começa a subir, antes que o aroma chegue. Não se faz a bênção antes que a fumaça suba.

13. Se o "Mugmar" for de madeira, a bênção é "Borê Atzei Besamim"; se for de erva, "Borê Isvei Besamim"; se for de outros tipos, "Borê Minei Besamim".

14. Quem cheira o etrog de mitzvá faz a bênção sobre ele. Alguns dizem que não se faz a bênção, portanto, é aconselhável não cheirá-lo. Há quem diga que quem cheira pão quente deve fazer a bênção "Hanoten Reiach Tov Bapat" (Que dá um bom aroma ao pão), mas outros dizem que não se deve fazer a bênção, portanto, é melhor não cheirá-lo.


--- EXPLICAÇÃO PRÁTICA ---

O Siman 216 do Shulchan Aruch Orach Chayim trata das leis relacionadas às bênçãos feitas sobre aromas agradáveis. Vamos entender isso de forma prática:

1. **Bênção Antes de Cheirar**: Antes de desfrutar de um bom aroma, é necessário recitar uma bênção específica. Isso é semelhante a fazer uma bênção antes de comer, reconhecendo a bondade de Deus em nos proporcionar prazeres sensoriais.

2. **Tipos de Bênçãos para Diferentes Fontes de Aroma**:
   - **Árvores Aromáticas**: Se o aroma vem de uma árvore, a bênção é "Borê Atzei Besamim" (Criador das árvores aromáticas).
   - **Ervas Aromáticas**: Se vem de uma erva, a bênção é "Borê Isvei Besamim" (Criador das ervas aromáticas).
   - **Outros Aromas**: Para aromas que não são nem de árvores nem de ervas, como almíscar, a bênção é "Borê Minei Besamim" (Criador de tipos de aromas).
   - **Frutas Aromáticas**: Se o aroma vem de uma fruta que se pretende cheirar, a bênção é "Hanoten Reiach Tov Baperot" (Que dá um bom aroma aos frutos). Se a fruta é apenas para comer, não se faz a bênção, mesmo que tenha um bom aroma.

3. **Bênção Genérica**: Em caso de dúvida sobre qual bênção usar, "Borê Minei Besamim" é uma opção segura, pois cobre todos os tipos de aromas.

4. **Exemplos Específicos**:
   - **Rosas e Canela**: A bênção é "Borê Atzei Besamim".
   - **Óleo de Bálsamo**: A bênção é "Borê Shemen Arev" (Criador do óleo agradável).
   - **Óleo de Oliva Aromático**: A bênção é "Borê Atzei Besamim".

5. **Óleos Perfumados**: Dependendo dos ingredientes, a bênção pode variar. Se o óleo contém tanto árvores quanto ervas, "Borê Minei Besamim" é apropriado. Se o aroma foi removido, pode haver dúvida sobre a bênção, então é melhor evitar cheirá-lo.

6. **Plantas e Flores Específicas**:
   - **Alecrim ou Jasmim**: A bênção é "Borê Atzei Besamim".
   - **Violeta**: A bênção é "Borê Isvei Besamim".
   - **Narciso**: A bênção depende de onde cresce; no jardim, "Borê Atzei Besamim", no campo, "Borê Isvei Besamim".

7. **Vários Aromas Juntos**: Se você tem diferentes tipos de aromas, faz-se a bênção apropriada para cada um. "Borê Minei Besamim" pode ser usada para todos se necessário.

8. **Mugmar (Incenso)**: A bênção é feita quando a fumaça começa a subir, antes de sentir o aroma.

9. **Aromas de Madeira ou Erva no Mugmar**: A bênção varia conforme a origem do aroma.

10. **Etrog e Pão**: Há discussões sobre se se deve fazer bênçãos sobre o aroma do etrog (fruta usada em Sucot) e pão quente. Para evitar dúvidas, é melhor não cheirá-los.

Essas leis mostram como o judaísmo valoriza a consciência e a gratidão por todas as experiências sensoriais, incluindo o olfato. Na prática, isso significa que, ao encontrar um aroma agradável, um judeu deve parar e reconhecer essa experiência com uma bênção apropriada.
