SIMAN 372

**Leis sobre parceiros de residências para o Eiruv. Contém 19 seções:**

1. Telhados, pátios e áreas cercadas são considerados uma única área para utensílios que permaneceram neles durante o Shabat. É permitido transportar de um para o outro, mesmo que pertençam a diferentes proprietários, e mesmo que não tenham feito um Eiruv juntos. É permitido transportar de um pátio para outro, ou para um telhado, ou para o topo de uma parede entre eles, e de um telhado para outro adjacente, mesmo que seja muito mais alto, e do telhado para uma área cercada que não exceda dois seah, ou que tenha sido cercada para habitação, mesmo que exceda dois seah, ou para um beco que tenha sido corrigido com uma viga ou uma trave, mesmo que não tenham feito um Eiruv nele. Mesmo que os moradores do pátio tenham feito um Eiruv entre si, pois é comum que utensílios das casas estejam no pátio, é permitido transportar utensílios que permaneceram no pátio para outro pátio, e não tememos que também transportem utensílios que permaneceram na casa para outro pátio, telhado ou área cercada.

2. Uma área cercada maior que dois seah que não foi cercada para habitação é considerada um carmelit, e é proibido transportar de uma para outra, exceto por dois cúbitos em cada uma.

3. Duas cortes que desejam fazer um Eiruv juntas para permitir até mesmo utensílios que permaneceram nas casas não precisam de outro Eiruv, mas o Eiruv que já fizeram pode ser levado por um dos moradores do pátio para todos e colocado em uma das casas do segundo pátio. Se desejar, pode até levar seu próprio pão e todos estarão permitidos. Nota: Isso é válido se os moradores do pátio fizeram um Eiruv com ele primeiro, ou se não precisam fazer um Eiruv, como explicado no Siman 370.

4. Duas cortes não podem fazer um Eiruv juntas a menos que haja uma abertura entre elas ou uma janela de quatro por quatro tefachim, e parte dela esteja dentro de dez tefachim do chão. Então, se desejarem, podem fazer um Eiruv juntas, ou cada uma por si. (Veja também Siman 376, Seção 2, sobre como os israelitas que vivem em dois ou três lugares podem fazer um Eiruv).

5. Uma janela entre duas casas, mesmo que esteja acima de dez tefachim, se desejarem, podem fazer um Eiruv juntas. O mesmo se aplica a uma abertura entre uma casa e um sótão, mesmo que esteja acima de dez tefachim, se desejarem, podem fazer um Eiruv juntas, mesmo que não haja escada para subir. Nota: No entanto, é necessário que tenha quatro por quatro (Beit Yosef). Se colocarem uma grade na frente da janela, ela perde o status de janela.

6. Uma casa entre dois pátios que está aberta para ambos e fez um Eiruv com ambos, mas os pátios não fizeram um Eiruv entre si, não é permitido transportar utensílios que permaneceram nas casas de um pátio para o outro através desta casa intermediária. Se houver uma parede entre dois pátios com dez tefachim de altura, ou se o solo de um dos pátios for cinco tefachim mais alto que o outro e uma barreira de cinco tefachim for colocada sobre ele para completar dez, não podem fazer um Eiruv juntos. Se a parede for alta para ambos dez tefachim e houver frutas no topo, os moradores dos dois pátios podem descê-las para seus pátios, e também podem subir frutas que permaneceram nos pátios para o topo e descê-las para outro pátio, seja a parede larga quatro tefachim ou não. Mas frutas que permaneceram na casa, se a parede for larga quatro tefachim, é proibido subir ou descer. Se não for larga quatro tefachim, é permitido. Nota: Desde que não troquem, como mencionado acima no Siman 346, Seção 1. Se a parede for alta para um dos pátios dez tefachim e para o outro não, como quando o solo de um é mais alto que o do outro, aquele para quem não é alto dez tefachim pode usá-lo, mesmo com utensílios que permaneceram na casa, e o outro é proibido de usá-lo, mesmo com utensílios que permaneceram no pátio. Isso é válido se a parede for larga quatro tefachim, mas se não for, é permitido para ambos usá-la, mesmo das casas, mesmo que cada um tenha feito um Eiruv por si.

7. Se a parede for aberta até dez cúbitos, é considerada uma porta. Nota: Desde que não esteja completamente aberta. Se estiver aberta mais de dez, é considerada um único pátio e precisam fazer um Eiruv juntos. (E certamente se não houver uma barreira de dez tefachim entre eles, precisam fazer um Eiruv juntos).

8. Uma parede entre dois pátios com dez tefachim de altura para ambos, e uma escada de quatro tefachim de largura foi colocada ao lado da parede, e também no outro pátio, mesmo que as escadas não estejam alinhadas, se não houver três tefachim entre elas, são consideradas uma porta. Se estiverem afastadas três tefachim, se a parede for larga quatro tefachim, ainda são consideradas uma porta. Se não for larga quatro tefachim, não são consideradas uma porta. Em que tipo de escada disseram isso? Quando tem quatro degraus, mas menos que isso não, a menos que seus braços e degraus sejam pesados, pois então seu peso a fixa. Se a parede não for alta, mas apenas dez tefachim, e uma escada de sete e algo mais de altura e quatro de comprimento for colocada ao lado da parede, permite usá-la, seja para usar sobre ela ou para fazer um Eiruv juntos, pois não resta três até o topo da parede. Se uma camada de terra for removida do topo da parede para reduzi-la de dez tefachim de altura, se tiver quatro de comprimento, é eficaz tanto para considerá-la uma porta para fazer um Eiruv juntos quanto para permitir o uso de toda a parede. Se não tiver quatro, não é considerada uma porta para fazer um Eiruv juntos, nem é eficaz para permitir o uso de toda a parede, pois só se pode usar a parte onde foi reduzida.

9. Se uma plataforma for construída na base da parede para reduzi-la de dez tefachim de altura, se tiver quatro de comprimento ao longo da parede e se projetar quatro, é eficaz para permitir o uso de toda a parede, mas não é considerada uma porta para fazer um Eiruv juntos até que alcance o topo da parede. Se não tiver quatro por quatro, não é eficaz nem mesmo para usar em frente a ela.

10. Se uma tigela de quatro por quatro e de qualquer altura for virada para reduzir a altura de dez tefachim, é permitido usar em frente à redução, desde que seja fixada com argamassa.

11. Se uma plataforma for construída sobre outra plataforma, se a inferior tiver quatro por quatro ou não tiver, mas a superior tiver quatro por quatro e não houver três tefachim entre elas, é eficaz para permitir o uso sobre ela, mas não é eficaz para ser uma porta para fazer um Eiruv juntos.

12. Se houver uma saliência saindo da parede com quatro por quatro e uma escada de qualquer tamanho for colocada sobre ela, é eficaz para permitir o uso da parede, desde que o degrau inferior não esteja a mais de três tefachim do chão e não haja três tefachim entre os degraus, e que a escada seja colocada sobre a saliência. Se for apenas apoiada ao lado, não é eficaz. Enquanto a parede não tiver vinte tefachim de altura, uma única saliência colocada no meio é suficiente, pois não há dez tefachim de altura até a saliência, nem dez tefachim de altura dela até o topo da parede. Se tiver vinte tefachim de altura, são necessárias duas saliências, uma dentro dos dez tefachim inferiores e outra acima dela dentro dos dez tefachim superiores, pois se a saliência estiver a dez tefachim de altura, uma escada de qualquer tamanho não é suficiente. As saliências não devem estar alinhadas, para que seja possível colocar uma escada de uma para a outra.

13. Se duas escadas forem colocadas lado a lado e nenhuma delas tiver quatro de comprimento, mas forem afastadas para que juntas tenham quatro, e o espaço entre elas for preenchido com palha, não é eficaz para permitir o uso sobre ela nem para fazer um Eiruv juntos, pois o local onde os pés se apoiam é no meio da escada e não é adequado subir pela palha. Se a escada for colocada no meio e a palha ao lado, é eficaz tanto para permitir o uso sobre ela quanto para fazer um Eiruv juntos.

14. Se a escada não tiver quatro de largura e uma cavidade for feita ao lado dela na parede para completá-la a quatro, é suficiente cavar até a altura de dez. Se nenhuma escada for colocada, mas uma cavidade for feita na parede como degraus de uma escada para subir, é necessário cavar em toda a altura da parede, e é eficaz tanto para permitir o uso quanto para fazer um Eiruv juntos.

15. Se houver uma árvore ao lado da parede e ela for usada como escada para a parede, se desejarem, podem fazer um Eiruv juntos. Mas se uma árvore de idolatria for usada como escada para a parede, não podem fazer um Eiruv juntos, pois é proibido subir nela pela Torá, já que é proibida para uso. O Rosh escreveu o contrário, que uma árvore não é eficaz, mas uma árvore de idolatria é eficaz, desde que esteja seca.

16. Se houver uma vala entre dois pátios com dez tefachim de profundidade e quatro de largura, não podem fazer um Eiruv juntos, mesmo que esteja cheia de palha e feno, enquanto não for anulada. Se estiver cheia de terra e pedras, mesmo que não tenha sido anulada, precisam fazer um Eiruv juntos. (Veja também Siman 358, Seção 2). Nota: Alguns dizem que se estiver cheia de frutas, é considerada anulada, pois é comum usá-la como depósito.

17. Se uma tábua for colocada sobre ela como uma ponte de uma borda da vala à outra, se tiver quatro de largura, é considerada uma porta e podem fazer um Eiruv juntos, ou, se desejarem, podem fazer dois Eiruvim separados. Nota: O mesmo se aplica se for preenchida com algo de quatro de largura que seja anulado ali. Se a tábua for colocada ao longo da vala com quatro de comprimento, mesmo que seja de qualquer tamanho, é considerada uma porta, pois a reduziu de quatro.

18. Se a vala for profunda para um dos lados dez tefachim e para o outro não, ou se for nivelada para ambos, é considerada como uma parede.

19. Se houver um monte de feno entre dois pátios, se tiver dez tefachim de altura, cada um faz seu próprio Eiruv. Se for reduzido durante a semana para menos de dez, precisam fazer um Eiruv juntos. Enquanto não for reduzido, é proibido para qualquer um deles colocar feno em sua cesta no Shabat para alimentar seu animal, e mesmo colocar o animal ali para comer é proibido. Mas se puder ficar na frente dela para que não tenha outro caminho a não ser para lá, é permitido. Segundo o Rosh, mesmo durante a semana é proibido colocar feno em sua cesta ou colocar o animal ali com as mãos. Isso se aplica a um monte entre dois pátios, mas um monte entre duas casas é permitido alimentar seu animal com as mãos.


--- EXPLICAÇÃO PRÁTICA ---

Para entender as leis do Siman 372 do Shulchan Aruch Orach Chayim, é importante primeiro compreender o conceito de "Eiruv". Um Eiruv é uma solução haláchica que permite transportar objetos em áreas públicas ou compartilhadas durante o Shabat, quando normalmente seria proibido. Ele cria uma "unidade residencial" simbólica entre diferentes domicílios, permitindo que os moradores compartilhem espaços comuns como se fossem uma única residência.

Aqui está uma explicação prática dos principais pontos abordados no texto:

1. **Áreas Compartilhadas**: Telhados, pátios e áreas cercadas são considerados uma única área para objetos que já estavam lá quando o Shabat começou. Isso significa que é permitido mover esses objetos entre essas áreas, mesmo que pertençam a diferentes proprietários e não tenham feito um Eiruv. Isso é relevante em comunidades onde várias famílias compartilham um espaço comum.

2. **Carmelit**: Uma área cercada maior que dois seah (uma medida de área) que não foi cercada para habitação é considerada um "carmelit", uma categoria de espaço que não é nem totalmente público nem privado. Transportar objetos entre essas áreas é restrito, exceto por uma pequena distância.

3. **Fazendo um Eiruv**: Duas cortes (ou pátios) que desejam fazer um Eiruv juntas podem usar o Eiruv já existente de uma delas, colocando-o em uma casa do outro pátio. Isso simplifica o processo de criar um Eiruv entre áreas adjacentes.

4. **Aberturas e Janelas**: Para que duas cortes façam um Eiruv juntas, deve haver uma abertura ou janela específica entre elas. Isso simboliza a conexão entre os espaços, permitindo que sejam considerados uma única unidade residencial.

5. **Casas Intermediárias**: Se uma casa está entre dois pátios e fez um Eiruv com ambos, mas os pátios não fizeram um Eiruv entre si, não é permitido transportar objetos de um pátio para o outro através dessa casa.

6. **Paredes e Escadas**: A altura e a largura das paredes entre pátios, bem como a presença de escadas, influenciam se os pátios podem ser considerados conectados para fins de Eiruv. Detalhes como a altura da parede e a largura das escadas determinam se é possível fazer um Eiruv conjunto.

7. **Valas e Montículos**: Valas e montículos entre pátios também afetam a possibilidade de fazer um Eiruv. A profundidade e a largura da vala, ou a altura do montículo, determinam se os pátios são considerados separados ou conectados.

Essas leis são aplicadas principalmente em comunidades onde várias famílias vivem próximas e compartilham espaços comuns. O Eiruv facilita a vida no Shabat, permitindo que as pessoas transportem objetos necessários sem violar as restrições do dia sagrado. Em contextos urbanos modernos, muitas comunidades estabelecem Eiruvim para permitir que os moradores se movimentem livremente entre suas casas e áreas públicas durante o Shabat.
