SIMAN 276

**Leis sobre uma vela acesa por um não-judeu no Shabat. Contém cinco seções:**

1. Se um não-judeu acender uma vela para um judeu, é proibido para qualquer judeu, mesmo para aquele para quem não foi acesa. *Nota: Não há diferença se o não-judeu foi pago ou não, ou se o trabalho foi feito por contrato ou aluguel, pois, já que o judeu se beneficia do trabalho realizado no Shabat, é proibido em qualquer caso.*

2. No entanto, se o não-judeu acender a vela para seu próprio uso ou para um judeu doente, mesmo que não esteja em perigo, é permitido a qualquer judeu usar sua luz. *Nota: Isso também se aplica se ele fizer uma fogueira para seu próprio uso ou para um doente. Alguns proíbem a fogueira, temendo que ele possa aumentar o fogo para o judeu. Contudo, se o não-judeu fizer isso na casa de um judeu por sua própria iniciativa, o judeu não precisa sair, mesmo que se beneficie da vela ou da fogueira.*

3. Se um judeu e um não-judeu estiverem juntos e o não-judeu acender uma vela, se a maioria for de não-judeus, é permitido usar sua luz. Se a maioria for de judeus ou mesmo metade a metade, é proibido. Se houver evidência de que ele acendeu para seu próprio uso, como se o virmos usando a luz, mesmo que a maioria seja de judeus, é permitido. *Nota: Alguns dizem que é permitido pedir a um não-judeu para acender uma vela para a refeição de Shabat, pois acreditam que é permitido pedir a um não-judeu para realizar um trabalho completo em caso de mitzvá. Com base nisso, muitos são lenientes em pedir a um não-judeu para acender velas para uma refeição, especialmente em um casamento ou brit milá, e não se deve impedi-los. No entanto, deve-se ser rigoroso quando não há grande necessidade, pois a maioria dos decisores discorda dessa opinião.*

4. Se alguém disser ao seu servo ou serva para ir com ele e eles acenderem a vela, mesmo que também precisem dela, não é considerado para o uso do não-judeu, pois a principal razão para ir é para o judeu. *Nota: É permitido dizer a um não-judeu para ir com ele e pegar uma vela já acesa, pois ele está apenas carregando a vela.*

5. Se houver uma vela na casa de um judeu e um não-judeu acender outra vela, é permitido usar sua luz enquanto a primeira vela estiver acesa. Mas, depois que a primeira se apagar, é proibido usar a luz da segunda. Da mesma forma, se ele adicionar óleo a uma vela acesa, é permitido usar até que o óleo original se esgote, e depois é proibido. *Nota: É permitido impedir um não-judeu de acender uma vela ou adicionar óleo.*

6. Em países frios, é permitido que um não-judeu faça uma fogueira para as crianças, e os adultos podem se aquecer nela. Mesmo para os adultos, é permitido se o frio for intenso, pois todos são considerados doentes em relação ao frio. Isso não se aplica àqueles que permitem mesmo quando o frio não é intenso naquele dia.


--- EXPLICAÇÃO PRÁTICA ---

O Siman 276 do Shulchan Aruch Orach Chayim aborda as regras sobre o uso de luz ou fogo aceso por um não-judeu durante o Shabat. Vamos explicar cada seção de forma prática e acessível:

1. **Vela acesa por um não-judeu para um judeu**: Se um não-judeu acender uma vela especificamente para um judeu no Shabat, nenhum judeu pode usar essa luz. Isso se aplica independentemente de o não-judeu ter sido pago ou não. A razão é que o Shabat é um dia de descanso, e os judeus não devem se beneficiar de trabalhos realizados por outros em seu favor durante esse dia.

2. **Vela acesa para uso próprio ou para um judeu doente**: Se o não-judeu acender a vela para si mesmo ou para um judeu que está doente (mas não em perigo de vida), qualquer judeu pode usar essa luz. Isso também se aplica a uma fogueira. A ideia é que, se o não-judeu acendeu a luz para seu próprio benefício, não há problema em um judeu também se beneficiar incidentalmente.

3. **Judeus e não-judeus juntos**: Se um não-judeu acende uma vela em um grupo misto, a permissão para usar a luz depende de quem é a maioria. Se a maioria for de não-judeus, é permitido usar a luz. Se a maioria for de judeus ou igual, é proibido, a menos que fique claro que o não-judeu acendeu a vela para si mesmo.

4. **Instruções para um servo ou serva**: Se um judeu pede a um servo ou serva para acompanhá-lo e eles acendem uma vela, isso é considerado como feito para o judeu, mesmo que o servo também precise da luz. No entanto, é permitido pedir a um não-judeu para carregar uma vela já acesa.

5. **Vela adicional acesa por um não-judeu**: Se já houver uma vela acesa na casa de um judeu e um não-judeu acender outra, é permitido usar a luz da segunda vela enquanto a primeira ainda estiver acesa. Após a primeira vela se apagar, não é mais permitido usar a luz da segunda. O mesmo se aplica se o não-judeu adicionar óleo a uma vela acesa; é permitido usar até que o óleo original se esgote.

6. **Fogueira em países frios**: Em lugares frios, é permitido que um não-judeu faça uma fogueira para aquecer crianças, e os adultos também podem se aquecer. Se o frio for intenso, todos são considerados como se estivessem doentes em relação ao frio, permitindo que se aqueçam na fogueira.

Essas regras refletem o equilíbrio entre manter o descanso do Shabat e as necessidades práticas da vida, especialmente em situações de necessidade ou conforto básico.
