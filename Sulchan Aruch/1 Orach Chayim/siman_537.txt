SIMAN 537

**Lei sobre o trabalho para evitar perda. Contém 16 seções:**

1. É permitido realizar um trabalho que evite perda durante o Chol HaMoed (dias intermediários de Pessach e Sucot) sem alteração. No entanto, deve-se minimizar o esforço sempre que possível (Kol Bo). Portanto, um campo irrigado (ou seja, uma terra sedenta) que começou a ser regado antes do festival pode continuar a ser regado, pois se não for regado agora, será danificado. Mas um campo de sequeiro, que não é regado exceto para melhorar, é proibido regar.

2. Mesmo em um campo irrigado, só é permitido onde não há esforço excessivo, como quando a água é conduzida de uma fonte, seja nova ou antiga, mas não se pode tirar água de um reservatório ou de águas pluviais acumuladas, pois isso é um grande esforço. Mesmo em um canteiro, se metade for alta e metade baixa, não se tira água da parte baixa para regar a parte alta.

3. Rios que fluem de lagos podem ser usados para regar um campo irrigado durante o festival, desde que não tenham parado de fluir. Da mesma forma, reservatórios por onde passa um canal de água podem ser usados para regar. E um reservatório que transbordou de um campo irrigado e ainda está transbordando pode ser usado para regar outro campo irrigado, desde que a fonte que rega o campo superior não tenha parado.

4. Vegetais que se deseja consumir durante o festival podem ser regados para que cresçam e estejam prontos para o festival. Mas se não se pretende consumi-los durante o festival e se rega apenas para melhorá-los, é proibido.

5. Não se fazem sulcos na base das videiras para que se encham de água. Se já estavam feitos e se deterioraram, podem ser consertados durante o festival, mas não se pode aprofundá-los mais do que eram originalmente (Magid, capítulo 8).

6. Um canal de água que se deteriorou pode ser consertado. Se tinha um côvado de profundidade, pode-se cavar até seis côvados. Se tinha dois côvados de profundidade, pode-se aprofundar até sete.

7. Pode-se conduzir água de uma árvore para outra, desde que não se regue todo o campo. Se o campo for úmido, é permitido regar todo ele.

8. É proibido abrir um local no campo para que a água entre e o regue. Se for feito para capturar peixes para consumo durante o festival, como abrir um local acima para que entrem e abaixo para que saiam, é permitido.

9. É proibido nivelar o campo para arar. Se for evidente que se está nivelando para debulhar durante o festival, como nivelar todo o campo, é permitido.

10. É proibido recolher madeira do campo para prepará-lo para arar. Se for evidente que se está recolhendo para uso próprio, como pegar as grandes e deixar as pequenas, é permitido.

11. É proibido cortar galhos da árvore para consertá-la. Se for evidente que se está cortando para alimentar os animais, como cortar todos de um lado, é permitido.

12. Não se pode aplicar pesticidas nas árvores nem fertilizar as plantações (ou seja, aplicar esterco para que a árvore não morra), mas pode-se untar as árvores e os frutos com óleo.

13. Ratos e camundongos que danificam o campo de árvores podem ser capturados da maneira usual, cavando um buraco e colocando uma armadilha, mesmo em um campo de cereais próximo ao campo de árvores, pois saem de lá e danificam as árvores. Se não estiver próximo ao campo de árvores, só se pode capturá-los com uma alteração, como fincar uma estaca no chão e movê-la até formar um buraco e colocar a armadilha. Alguns dizem que em um campo de cereais próximo ao campo de árvores só é permitido capturar com alteração, e se não estiver próximo, é proibido mesmo com alteração.

14. Não se pode colocar ovelhas no curral durante o festival para fertilizar o campo. Se um não-judeu as colocou por conta própria, é permitido, mesmo no Shabat, desde que não se pague a ele, nem mesmo pelo alimento. No Yom Tov, pode-se pagar pelo alimento, desde que não se pague mais nada. Durante o festival, mesmo que se pague mais, é permitido, desde que não se alugue, não se ajude, nem se entregue um pastor para mover as ovelhas (ou seja, levá-las de um lugar para outro para que o esterco se espalhe). Se o não-judeu for um trabalhador contratado por semana, mês ou ano, pode-se ajudá-lo durante o festival e entregar-lhe um pastor para mover as ovelhas.

15. É proibido ceifar o campo durante o festival se não houver perda ao esperar até depois do festival. Se não tiver o que comer, mesmo que encontre no mercado para comprar, não é obrigado a comprar, mas pode ceifar, juntar, debulhar, joeirar e peneirar da maneira usual, desde que não debulhe com animais. Isso se aplica quando precisa apenas para si mesmo, mas se precisar debulhar para muitos, pode debulhar mesmo com animais.

16. Quem tem uma vinha ao lado de uma vinha de um não-judeu, e o não-judeu colhe a sua durante o festival, se o judeu não colher a sua também, ele perderá. Pode colher, pisar as uvas, fazer os barris e todas as necessidades do vinho sem alteração, desde que não tenha a intenção de fazer seu trabalho durante o festival.


--- EXPLICAÇÃO PRÁTICA ---

O Siman 537 do Shulchan Aruch Orach Chayim trata das regras sobre o que é permitido fazer durante o Chol HaMoed, que são os dias intermediários dos festivais judaicos de Pessach e Sucot. Durante esses dias, algumas atividades de trabalho são restritas, mas há exceções para evitar perdas financeiras significativas. Vamos explicar cada seção de forma prática:

1. **Trabalho para evitar perda**: Durante o Chol HaMoed, é permitido realizar trabalhos que evitem perdas financeiras, como regar um campo que precisa de água para não ser danificado. No entanto, deve-se tentar minimizar o esforço envolvido.

2. **Esforço excessivo**: Mesmo em situações onde o trabalho é permitido, como regar um campo, não se deve fazer um esforço excessivo. Por exemplo, usar água de uma fonte é permitido, mas não se deve usar métodos que exijam muito trabalho, como tirar água de um reservatório manualmente.

3. **Uso de rios e reservatórios**: Se um rio ou reservatório está naturalmente fluindo, pode-se usar essa água para regar campos, desde que o fluxo não tenha parado.

4. **Regar vegetais**: Se você pretende consumir vegetais durante o festival, pode regá-los para que cresçam. Se não for consumi-los durante o festival, regar apenas para melhorar a colheita futura é proibido.

5. **Sulcos nas videiras**: Não se deve fazer novos sulcos para água nas videiras, mas se já existiam e se deterioraram, podem ser consertados, desde que não sejam aprofundados além do original.

6. **Conserto de canais de água**: Canais de água que se deterioraram podem ser consertados, mas há limites para o quanto se pode aprofundá-los.

7. **Conduzir água entre árvores**: É permitido conduzir água de uma árvore para outra, mas não se deve regar todo o campo, a menos que o campo já seja naturalmente úmido.

8. **Captura de peixes**: Não se deve abrir canais de água no campo para irrigação, mas se for para capturar peixes para consumo durante o festival, é permitido.

9. **Nivelamento de campo**: Não se deve nivelar um campo para arar, mas se for evidente que é para debulhar durante o festival, é permitido.

10. **Recolher madeira**: Não se deve recolher madeira do campo para prepará-lo para arar, mas se for para uso próprio, como pegar as grandes e deixar as pequenas, é permitido.

11. **Cortar galhos**: Não se deve cortar galhos para consertar a árvore, mas se for para alimentar animais, é permitido.

12. **Cuidados com árvores**: Não se deve aplicar pesticidas ou fertilizantes, mas pode-se passar óleo nas árvores e frutos.

13. **Captura de roedores**: Ratos e camundongos que danificam campos podem ser capturados, mas há regras sobre como isso deve ser feito, dependendo da proximidade com campos de árvores.

14. **Uso de ovelhas para fertilização**: Não se deve colocar ovelhas no curral para fertilizar o campo, mas se um não-judeu fizer isso por conta própria, é permitido, com algumas condições.

15. **Ceifar o campo**: Não se deve ceifar o campo durante o festival se não houver perda ao esperar. No entanto, se não houver comida suficiente, é permitido ceifar e processar o grão, mas com algumas restrições.

16. **Colheita ao lado de um não-judeu**: Se um não-judeu está colhendo sua vinha durante o festival e isso pode causar perda ao judeu, ele pode colher sua vinha também, mas sem a intenção de fazer isso especificamente durante o festival.

Essas regras ajudam a equilibrar a observância religiosa com a necessidade prática de evitar perdas financeiras significativas durante os dias intermediários dos festivais.
