SIMAN 311

**Leis sobre um falecido no Shabat e outras formas de movimentação indireta. Contém nove seções:**

1. Um falecido que está em um local onde há risco de incêndio pode ser movido usando um pão ou uma criança. Se não houver pão ou criança, e houver duas camas, pode-se movê-lo virando-o de uma cama para outra, o que é considerado movimentação indireta. Se não houver nem uma coisa nem outra, pode-se movê-lo completamente, mas isso é permitido apenas dentro do mesmo domínio.

2. Um falecido exposto ao sol pode ser movido da luz para a sombra no mesmo domínio usando um pão ou uma criança. Se não houver pão ou criança, não se deve movê-lo de forma alguma, nem mesmo virando-o de uma cama para outra, pois a movimentação indireta ainda é considerada movimentação.

3. Um falecido que começou a cheirar mal em uma casa, causando constrangimento aos vivos, pode ser removido para um carmelit (um tipo de domínio público rabínico). Se houver uma saída disponível, não se deve removê-lo, mas sim deixar o local. Alguns dizem que só é permitido removê-lo para um carmelit com um pão ou uma criança. Outros opinam que é melhor removê-lo sem pão ou criança para minimizar a movimentação. Há quem permita removê-lo até mesmo para um domínio público com uma criança, mas não com um pão. O mesmo se aplica se houver outra forma de desonra, como em um barco com gentios presentes.

4. É permitido pedir a um gentio para mover o falecido, assim como se fosse com um pão ou uma criança. No entanto, é proibido mover um falecido com um pão ou uma criança para o benefício de cohanim (sacerdotes) ou por qualquer outro motivo, mas alguns permitem que um gentio o faça. Vi que é costume permitir isso para uma mitzvá ou casamento.

5. Alguns dizem que se colocar uma peça de roupa importante sobre o falecido, é como se tivesse colocado um pão ou uma criança.

6. Alguns dizem que a necessidade de um pão ou criança se aplica apenas a um falecido nu. Se ele estiver vestido, não é necessário um pão ou uma criança.

7. Se precisar do local onde o falecido está ou do objeto sobre o qual ele está deitado, é permitido movê-lo indiretamente, ou seja, virando-o de uma cama para outra, pois é para um propósito permitido. Não se permite mover com um pão ou uma criança, exceto no caso de um falecido, mas não para outros objetos proibidos de serem movidos.

8. Se um falecido estiver exposto ao sol e não houver lugar para movê-lo, ou se não quiserem movê-lo, duas pessoas podem sentar-se de cada lado dele. Se estiver quente embaixo, cada um traz sua cama e senta-se sobre ela. Se estiver quente em cima, cada um traz uma esteira e a coloca sobre ele. Então, cada um levanta sua cama e sai, formando uma barreira naturalmente, pois as esteiras estão próximas uma da outra e suas extremidades estão no chão de cada lado do falecido.

9. É permitido ungir o falecido (e lavá-lo) e remover o travesseiro debaixo dele para que não cheire mal, desde que nenhum membro seja movido. Se a boca estiver se abrindo, pode-se amarrar o queixo para que não se abra mais, mas não para fechá-lo, pois isso moveria um membro. Por esse motivo, não se fecham os olhos de um falecido no Shabat.

10. Movimentação indireta para um propósito permitido é permitida. Portanto, um rabanete enterrado na terra, com parte exposta, que não criou raízes e não foi intencionado para plantio, pode ser retirado, mesmo que ao fazê-lo mova a terra. Mesmo que tenha absorvido umidade do solo, é permitido. Mesmo se foi deixado lá antes do Shabat para ficar o Shabat inteiro, não há problema, pois alimentos não são considerados base para algo proibido.

11. Movimentação com o corpo, mesmo para algo proibido, é permitida. Portanto, palha sobre a cama, que geralmente é reservada para combustível, pode ser movida com o corpo. Se um travesseiro ou almofada foi colocado sobre ela antes do Shabat, pode-se movê-la até com a mão, pois foi preparada para deitar-se sobre ela.

12. Frutas enterradas em palha ou feno, que são reservadas, podem ser retiradas espetando uma agulha ou espeto, e a palha cai sozinha. Se estiverem enterradas em areia ou terra, veja o Siman 338, Seção 38.


--- EXPLICAÇÃO PRÁTICA ---

O Siman 311 do Shulchan Aruch Orach Chayim trata de como lidar com um falecido no Shabat, um dia sagrado de descanso no judaísmo, quando muitas atividades são restritas. Vamos explicar os principais pontos de forma prática:

1. **Movimentação Indireta de um Falecido**: No Shabat, mover um falecido diretamente é geralmente proibido. No entanto, se houver um risco, como um incêndio, é permitido movê-lo de forma indireta, usando um objeto permitido, como um pão ou uma criança, para justificar a ação. Se não houver essas opções, pode-se movê-lo de uma cama para outra, o que é considerado uma forma indireta de movimentação.

2. **Proteção do Sol**: Se o falecido estiver exposto ao sol, pode-se movê-lo para a sombra usando um pão ou uma criança, mas não de outra forma. Isso é para evitar a movimentação direta, que é restrita no Shabat.

3. **Mau Cheiro e Constrangimento**: Se o corpo começar a cheirar mal, causando desconforto, ele pode ser movido para um carmelit, um tipo de espaço público rabínico. Há diferentes opiniões sobre como isso deve ser feito, mas a ideia é minimizar a movimentação direta.

4. **Pedir Ajuda a um Gentio**: É permitido pedir a um não-judeu para mover o falecido, o que é uma forma de contornar a proibição de movimentação direta no Shabat. Isso é especialmente relevante em situações de necessidade, como para uma mitzvá (um mandamento) ou um casamento.

5. **Uso de Roupas**: Colocar uma peça de roupa importante sobre o falecido pode ser considerado como usar um pão ou uma criança, permitindo a movimentação indireta.

6. **Falecido Vestido**: Se o falecido estiver vestido, não é necessário usar um pão ou uma criança para movê-lo, pois a roupa já serve como um meio indireto.

7. **Necessidade do Local ou Objeto**: Se precisar do espaço onde o falecido está, é permitido movê-lo indiretamente, como de uma cama para outra, mas não com um pão ou uma criança, exceto no caso de um falecido.

8. **Proteção do Calor**: Se o falecido estiver exposto ao calor, pode-se criar uma barreira ao redor dele usando camas e esteiras, sem movê-lo diretamente.

9. **Cuidados com o Corpo**: É permitido cuidar do corpo do falecido, como ungir ou lavar, desde que não se mova nenhum membro. Amarrar o queixo para evitar que a boca se abra é permitido, mas não para fechá-la.

10. **Movimentação Indireta Permitida**: Movimentar algo indiretamente para um propósito permitido é aceitável. Por exemplo, um rabanete parcialmente enterrado pode ser retirado, mesmo que mova a terra.

11. **Movimentação com o Corpo**: Usar o corpo para mover algo, mesmo que seja proibido, é permitido. Por exemplo, palha sobre a cama pode ser movida com o corpo.

12. **Frutas Enterradas**: Frutas enterradas em palha podem ser retiradas sem problemas, mas se estiverem em areia ou terra, há regras específicas a seguir.

Essas leis refletem o equilíbrio entre o respeito pelo Shabat e a dignidade do falecido, permitindo ações necessárias de forma indireta ou através de terceiros.
