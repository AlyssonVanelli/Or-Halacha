SIMAN 533

**Tradução do Shulchan Aruch Orach Chayim (Siman 533):**

**Trabalhos permitidos durante Chol HaMoed. Este capítulo contém cinco seções:**

1. É permitido moer farinha para as necessidades de Chol HaMoed, mesmo que a pessoa tenha intencionado realizar o trabalho durante o Moed. Também é permitido cortar árvores que estão conectadas ao solo e preparar cerveja, seja de tâmaras ou de cevada, para as necessidades do Moed. No entanto, se não for para as necessidades do Moed, é proibido. Contudo, não é necessário ser exato; pode-se fazer de forma generosa, e se sobrar, sobrou.

2. Mas não se deve agir de forma enganosa para moer ou fazer mais cerveja intencionalmente, ou se já possui farinha (ou cerveja antiga), não deve enganar dizendo que deseja usar esta. No entanto, se a pessoa tem pão, é permitido moer para fazer pão fresco, pois pão fresco é preferível. O mesmo se aplica à cerveja; se a nova for preferível sem engano, é permitido (como mencionado pelo Magid no capítulo 7).

3. É permitido arrancar linho, pois é adequado para cobrir alimentos, e arrancar lúpulo, que é adequado para fazer cerveja para as necessidades do Moed, e arrancar gergelim, cujas sementes secas podem ser consumidas imediatamente.

4. Frutas que estão parcialmente maduras e podem ser consumidas com dificuldade podem ser colhidas para consumo. Se a pessoa as colheu para comer e sobrou, e teme que estraguem se não forem conservadas, pode conservá-las para extrair sua umidade. No entanto, é proibido colhê-las inicialmente com a intenção de conservá-las. Se for algo que não estará disponível após o Moed, é permitido comprá-lo e conservá-lo.

5. Uma pessoa pode pescar peixes tanto quanto puder e salgá-los durante o Moed, pois é possível que os consuma durante o Moed, se os espremer com as mãos várias vezes até que amoleçam. É permitido pescá-los em público, de modo que todos saibam que é para as necessidades do Moed (como mencionado por Mordechai).

6. Todos os trabalhos que são para as necessidades do Moed, quando realizados, devem ser feitos discretamente pelos profissionais. Por exemplo, caçadores (de animais e aves), moleiros e aqueles que colhem para vender no mercado devem fazê-lo discretamente para as necessidades do Moed. Nota: É permitido ordenhar o gado mesmo que não seja para as necessidades do Moed, pois é considerado uma perda se não for feito (como mencionado no Terumat HaDeshen, Siman 153, e Beit Yosef).


--- EXPLICAÇÃO PRÁTICA ---

O Siman 533 do Shulchan Aruch Orach Chayim trata das atividades permitidas durante o "Chol HaMoed", que são os dias intermediários das festas judaicas de Pessach e Sucot. Durante esses dias, algumas atividades de trabalho são restritas, mas há exceções para necessidades específicas. Vamos entender cada seção de forma prática:

1. **Moagem de Farinha e Preparação de Cerveja**: Durante o Chol HaMoed, é permitido moer farinha e preparar cerveja se for para uso durante o próprio Moed. Isso significa que se você precisa desses itens para as refeições ou celebrações durante o Moed, pode produzi-los. Mesmo que você acabe fazendo mais do que o necessário, não há problema se sobrar. A ideia é que o trabalho seja feito para atender às necessidades imediatas do Moed.

2. **Evitar Enganos**: Não é correto fingir que você precisa de mais farinha ou cerveja apenas para poder trabalhar mais durante o Moed. Se você já tem o suficiente, não deve usar truques para justificar mais trabalho. No entanto, se o pão ou a cerveja frescos são realmente preferidos, é permitido prepará-los, pois a qualidade superior justifica o esforço.

3. **Colheita de Linho, Lúpulo e Gergelim**: É permitido colher linho, lúpulo e gergelim durante o Moed, pois eles têm usos imediatos, como cobrir alimentos ou fazer cerveja. Isso significa que se esses itens são necessários para o Moed, você pode colhê-los.

4. **Colheita de Frutas**: Frutas que estão quase maduras podem ser colhidas se forem consumidas durante o Moed. Se você colheu mais do que pode consumir e teme que estraguem, é permitido conservá-las. No entanto, não se deve colher frutas com a intenção inicial de conservá-las, a menos que não estejam disponíveis após o Moed.

5. **Pesca de Peixes**: Pescar e salgar peixes é permitido, pois há a possibilidade de consumi-los durante o Moed. A pesca deve ser feita de forma que fique claro que é para o Moed, como pescar em público.

6. **Trabalhos Profissionais**: Profissionais como caçadores, moleiros e vendedores devem realizar seu trabalho discretamente, apenas para atender às necessidades do Moed. Isso significa que, embora possam trabalhar, devem fazê-lo de maneira que não pareça um dia normal de trabalho. Ordenhar gado é permitido mesmo que não seja para o Moed, pois não ordenhar causaria perda.

Essas regras ajudam a equilibrar a celebração e o descanso durante o Chol HaMoed, permitindo que as necessidades essenciais sejam atendidas sem transformar esses dias em dias comuns de trabalho.
