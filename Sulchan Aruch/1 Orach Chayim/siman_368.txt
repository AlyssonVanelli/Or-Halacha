SIMAN 368

**Regra sobre o que fazer se, após a realização do eruv, este se tornar inválido. Contém cinco seções:**

1. Se fizeram um eruv e, posteriormente, o eruv se tornou inválido, e um dos moradores do pátio deseja consertá-lo, se ele deseja fazer o eruv com o mesmo tipo de alimento, mesmo que tenha acabado, não precisa informar aos outros. Mesmo que ele use o alimento dos outros para o eruv. Porém, se ele deseja usar um tipo diferente de alimento e o primeiro tipo acabou, ele deve informar aos outros se estiver usando o alimento deles. Se estiver usando o seu próprio alimento, não precisa informá-los. Se o primeiro tipo não acabou, mas apenas diminuiu, mesmo que ele use o alimento dos outros, não precisa informá-los.

2. Se novos moradores se juntaram ao pátio e a casa tem uma abertura para dois pátios, é necessário informar aos outros, pois talvez eles não queiram fazer o eruv com os novos moradores.

3. Qual é a medida inicial do eruv? Quando há dezoito ou menos pessoas, a medida é de um figo seco para cada um. Se houver mais de dezoito, mesmo que sejam mil ou mais, a medida é o alimento de duas refeições, que são dezoito figos secos, o que equivale a cerca de seis ovos. Alguns dizem que equivale a cerca de oito ovos.

4. Se a quantidade do eruv diminuiu após o início do primeiro Shabat e restou qualquer quantidade, ainda é válido para os outros Shabatot.

5. Se o pão do eruv ficou mofado e impróprio para consumo, é como se tivesse acabado completamente e é necessário fazer um novo eruv. Por isso, é costume fazer o eruv com matzá, que não mofa rapidamente [Kol Bo, Beit Yosef, Siman 394]. Além disso, pode-se guardá-la durante os dias de Pessach e manter o eruv durante todo o ano [opinião própria]. Isso é melhor do que fazer o eruv a cada Shabat, para que não se esqueçam de fazê-lo. No entanto, se desejarem fazer o eruv toda sexta-feira e comer o eruv a cada Shabat, têm permissão para fazê-lo. Veja adiante, Siman 394.


--- EXPLICAÇÃO PRÁTICA ---

Para entender o que está sendo discutido no Siman 368 do Shulchan Aruch Orach Chayim, é importante primeiro saber o que é um "eruv". No contexto judaico, um eruv é uma solução haláchica que permite que judeus observantes carreguem objetos em espaços públicos durante o Shabat, algo que normalmente não é permitido. O eruv é feito simbolicamente, geralmente com alimentos, para criar uma "área comum" entre as casas de um pátio ou bairro.

Aqui está uma explicação prática das regras mencionadas:

1. **Conserto do Eruv**: Se o eruv que foi feito se torna inválido (por exemplo, o alimento usado estraga ou é consumido), qualquer morador pode consertá-lo. Se ele usar o mesmo tipo de alimento que foi usado originalmente, não precisa avisar os outros moradores, mesmo que use o alimento deles. Mas se ele quiser usar um tipo diferente de alimento e o original acabou, ele deve informar aos outros se estiver usando o alimento deles. Se ele usar seu próprio alimento, não precisa avisar. Se o alimento original não acabou completamente, mas apenas diminuiu, ele pode usar o alimento dos outros sem avisar.

2. **Novos Moradores**: Se novas pessoas se mudam para o pátio e a casa tem acesso a dois pátios, é necessário informar aos outros moradores. Isso é porque eles podem não querer incluir os novos moradores no eruv.

3. **Medida do Eruv**: A quantidade de alimento necessária para o eruv depende do número de pessoas. Para até dezoito pessoas, é necessário um figo seco por pessoa. Se houver mais de dezoito pessoas, a quantidade total necessária é o equivalente a duas refeições, que são cerca de dezoito figos secos, ou aproximadamente seis a oito ovos, dependendo da opinião.

4. **Diminuição do Eruv**: Se a quantidade de alimento do eruv diminui após o primeiro Shabat, mas ainda resta alguma quantidade, o eruv continua válido para os Shabatot seguintes.

5. **Pão Mofado**: Se o pão usado no eruv fica mofado e impróprio para consumo, é como se o eruv tivesse acabado, e um novo deve ser feito. Por isso, é comum usar matzá, que não mofa rapidamente, para o eruv. A matzá pode ser guardada durante Pessach e usada para o eruv durante todo o ano. Isso é preferível a fazer um novo eruv a cada Shabat, para evitar esquecer de fazê-lo. No entanto, se alguém quiser fazer o eruv toda sexta-feira e comer o alimento do eruv a cada Shabat, isso também é permitido.

Essas regras são práticas para comunidades judaicas que desejam manter a observância do Shabat enquanto facilitam a vida cotidiana, permitindo o transporte de objetos em áreas públicas.
