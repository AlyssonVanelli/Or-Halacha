SIMAN 42

**Se é permitido mudar os tefilin do braço para os da cabeça. E contém três seções:**

1. É proibido mudar os tefilin da cabeça para fazer deles tefilin do braço. E mesmo pegar uma correia deles e colocá-la nos tefilin do braço é proibido, porque não se rebaixa de uma santidade maior para uma santidade menor, e a santidade dos tefilin da cabeça é maior, pois a maior parte do Nome de Deus está nos tefilin da cabeça. Mas mudar dos tefilin do braço para os da cabeça é permitido. E se forem novos, que ainda não foram usados, é permitido mudar mesmo dos tefilin da cabeça para os do braço, pois eles têm uma única cobertura de couro e parecem como uma única casa.

2. Se a pessoa fez uma condição desde o início, mesmo que já os tenha usado, pode mudá-los mesmo dos tefilin da cabeça para os do braço.

3. Um pano que foi designado para guardar tefilin para sempre, e foi usado uma vez para guardar tefilin, é proibido usá-lo para guardar dinheiro. Nota: Se a pessoa fez uma condição desde o início, em qualquer caso, é permitido (Beit Yosef em nome de Rabbeinu Yerucham e Mahari Aboab). E um pergaminho preparado para tefilin é proibido para escrever assuntos seculares, pois tal designação para um objeto de santidade é significativa (Ran no final do capítulo "Nigmar HaDin"). Se foi designado mas não foi usado, ou foi usado mas não foi designado, é permitido usá-lo para guardar dinheiro.


--- EXPLICAÇÃO PRÁTICA ---

Para entender o que está sendo discutido no Siman 42 do Shulchan Aruch Orach Chayim, é importante primeiro saber o que são os tefilin. Tefilin são pequenos recipientes de couro que contêm pergaminhos com passagens da Torá. Eles são usados por homens judeus durante as orações matinais nos dias de semana. Existem dois conjuntos de tefilin: um para o braço e outro para a cabeça.

### Explicação Prática:

1. **Mudança de Tefilin da Cabeça para o Braço:**
   - **Proibição de Rebaixamento de Santidade:** Os tefilin da cabeça são considerados de maior santidade do que os do braço. Isso se deve ao fato de que a maior parte do Nome de Deus está nos tefilin da cabeça. Portanto, não é permitido usar partes dos tefilin da cabeça (como as correias) para os tefilin do braço, pois isso seria considerado um rebaixamento de santidade.
   - **Exceção para Tefilin Novos:** Se os tefilin ainda não foram usados, é permitido mudar de um para o outro, pois eles ainda não foram designados especificamente para uma função.

2. **Condições Prévias:**
   - **Fazendo uma Condição:** Se, desde o início, a pessoa fez uma condição específica de que os tefilin poderiam ser usados de forma intercambiável, então é permitido mudar os tefilin da cabeça para os do braço, mesmo que já tenham sido usados.

3. **Uso de Objetos Associados aos Tefilin:**
   - **Pano para Guardar Tefilin:** Um pano que foi designado para guardar tefilin não pode ser usado para guardar dinheiro, pois foi separado para um propósito sagrado. No entanto, se uma condição foi feita desde o início permitindo seu uso para outros fins, então é permitido.
   - **Pergaminho para Tefilin:** Um pergaminho preparado para ser usado em tefilin não pode ser usado para escrever assuntos seculares, pois foi designado para um uso sagrado. Se o pergaminho foi designado mas não usado, ou usado mas não designado, ele pode ser usado para outros fins.

### Relevância Prática:

Essas leis refletem a importância de respeitar a santidade dos objetos usados em práticas religiosas. Na vida judaica contemporânea, isso se traduz em um cuidado especial com os objetos sagrados, garantindo que eles sejam usados de acordo com seu propósito e respeitando sua santidade. Além disso, a possibilidade de fazer condições prévias oferece flexibilidade, permitindo que as pessoas planejem o uso de seus objetos sagrados de acordo com suas necessidades, desde que respeitem as regras estabelecidas.
