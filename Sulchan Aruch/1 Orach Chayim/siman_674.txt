SIMAN 674

**É permitido acender de uma vela para outra. Este capítulo contém dois parágrafos:**

1. Pode-se acender uma vela de Chanucá a partir de outra vela de Chanucá, mas isso é permitido apenas diretamente, sem intermediários. No entanto, acender de uma vela de Chanucá através de uma vela comum é proibido. Há quem permita isso também, a menos que haja preocupação de que a vela comum se apague antes de acender outra vela de Chanucá. 

   *Nota: A prática é ser rigoroso com as velas de Chanucá, não acendendo nem mesmo de uma vela para outra, pois a principal obrigação é apenas uma vela, e as demais não são tão essenciais para a mitzvá. Portanto, não se deve acender de uma para outra. [Referências: Hagahot Maimoni e Mordechai] Tudo isso se aplica enquanto as velas estão acesas para cumprir a mitzvá, mas após o tempo da mitzvá, é permitido usufruir delas, e certamente é permitido acender a partir delas. [Beit Yosef, Hilchot Ketanot]*

2. Há quem diga que a vela da sinagoga, a vela de Shabat e a vela de Chanucá são todas para a mitzvá, e é permitido acender uma a partir da outra. 

   *Nota: O mesmo se aplica à vela para o estudo da Torá ou a uma vela para um doente que precisa de luz. [Nimukei Yosef, Hilchot Ketanot] [Sobre a vela da sinagoga, veja acima, Siman 154, Seif 14]*


--- EXPLICAÇÃO PRÁTICA ---

Para entender melhor o que está sendo discutido no Siman 674 do Shulchan Aruch Orach Chayim, vamos explicar os conceitos e práticas mencionados de forma acessível:

1. **Acender Velas de Chanucá:**
   - Durante a festa de Chanucá, é uma tradição judaica acender velas por oito noites para comemorar o milagre do óleo que durou mais do que o esperado no Templo Sagrado. Cada noite, uma vela adicional é acesa.
   - A questão aqui é sobre como acender essas velas: se é permitido usar uma vela de Chanucá já acesa para acender outra vela de Chanucá. A regra geral é que isso é permitido, mas deve ser feito diretamente, sem usar uma vela comum como intermediária.

2. **Uso de Velas Comuns:**
   - Usar uma vela comum para transferir a chama de uma vela de Chanucá para outra é geralmente proibido. Isso se deve ao respeito especial que se deve ter pelas velas de Chanucá, que são acesas para cumprir uma mitzvá (mandamento religioso).
   - No entanto, algumas opiniões permitem isso, desde que haja certeza de que a vela comum não se apagará antes de acender a próxima vela de Chanucá.

3. **Rigor na Prática:**
   - A prática mais rigorosa é não acender de uma vela de Chanucá para outra, mesmo diretamente, porque a obrigação principal é acender apenas uma vela por noite. As velas adicionais são consideradas menos essenciais, então é melhor evitar qualquer risco de desrespeito à mitzvá.

4. **Uso das Velas Após a Mitzvá:**
   - Após o tempo designado para cumprir a mitzvá, é permitido usar as velas de Chanucá para outros propósitos, incluindo acender outras velas.

5. **Outras Velas de Mitzvá:**
   - O texto também menciona que velas usadas para outras mitzvot, como a vela da sinagoga, a vela de Shabat (acendida para marcar o início do sábado) e velas usadas para estudar a Torá ou para um doente que precisa de luz, podem ser acesas umas a partir das outras. Isso se deve ao fato de todas serem usadas para cumprir mandamentos religiosos.

Essas regras refletem a importância de tratar as velas de Chanucá com respeito especial, reconhecendo seu papel na celebração de um milagre significativo na história judaica. Na prática contemporânea, essas diretrizes ajudam a manter a reverência e a intenção correta ao cumprir a mitzvá de acender as velas de Chanucá.
