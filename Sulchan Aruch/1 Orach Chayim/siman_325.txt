SIMAN 325

**Tradução do Shulchan Aruch Orach Chayim (Siman 325):**

**Um não-judeu que realiza trabalho para um judeu. Este capítulo contém dezesseis seções:**

1. É permitido convidar um não-judeu no Shabat. Também é permitido fornecer alimentos a ele no pátio para que os coma. Se ele os pegar e sair, não é necessário se preocupar, desde que o não-judeu esteja no pátio. Mas se ele estiver do lado de fora e estender a mão para dentro, sabendo-se que ele os levará para fora, ou se for para dar-lhe outros objetos que normalmente são levados para fora, é proibido, mesmo que ele esteja dentro, mesmo que os objetos pertençam ao não-judeu, pois quem vê não sabe que os objetos pertencem ao não-judeu. Nota: Mesmo que se tenha reservado um lugar para ele antes do Shabat, deve-se ser rigoroso (Mordechai, primeiro capítulo de Shabat).

2. Onde há preocupação com "caminhos de paz" ou com um não-judeu violento, é permitido dar ou enviar-lhe algo através de um não-judeu. O mesmo se aplica para uma necessidade de mitzvá, como remover o chametz de sua casa no Pessach (Mordechai).

3. É permitido trocar um penhor no Shabat se for uma vestimenta e ele a usar como vestimenta, pois isso não é considerado comércio. E também é permitido para um judeu, se ele precisar vesti-la. Nota: É melhor que o não-judeu pegue o penhor e coloque outro em seu lugar, e que o judeu não toque nele para que não pareça que está negociando (Agudah). Veja também no final do Siman 306 sobre a lei de um não-judeu que traz algo no Shabat, se é permitido recebê-lo.

4. Pão que um não-judeu assou para si mesmo no Shabat: há quem proíba e há quem permita. Em caso de necessidade ou para uma mitzvá, como uma refeição de brit milá ou para a bênção do hamotzi, pode-se confiar nos que permitem. Nota: Mas é proibido dar-lhe dinheiro antes do Shabat para que ele dê o pão no Shabat, pois então ele estaria fazendo isso em nome do judeu (Rabino Yerucham, Parte 1).

5. Um não-judeu que pescou peixes ou colheu frutas para si mesmo: é proibido para um judeu, mesmo que haja dúvida se ele os colheu ou pescou naquele dia, são proibidos naquele dia, mas à noite são permitidos imediatamente, mesmo que seja certo que ele os colheu ou pescou naquele dia.

6. Se ele colheu ou pescou para um judeu, ou para um judeu e um não-judeu, deve-se esperar até a noite o tempo suficiente para que pudessem ser feitos.

7. Se houver dúvida se ele os colheu para um judeu, ou se é sabido que ele os colheu para um judeu, mas não se sabe se foram colhidos naquele dia ou não, são proibidos naquele dia, e à noite deve-se esperar o tempo suficiente para que pudessem ser feitos. Há quem diga que à noite é permitido imediatamente.

8. Algo que não envolve preocupação de captura ou que está conectado, mas foi trazido de fora do limite: se o não-judeu trouxe para si mesmo, é permitido mesmo naquele dia. Se ele trouxe para um judeu, é permitido movê-lo, mesmo para quem foi trazido, mas comer é proibido naquele dia para quem foi trazido. À noite, deve-se esperar o tempo suficiente para que pudessem ser feitos. Nota: Há quem diga que a noite não conta no cálculo, e deve-se esperar no primeiro dia o tempo suficiente para que pudessem ser feitos [Semaq e Mordechai, final do capítulo]. Para outros, é permitido naquele dia. Há quem diga que para quem foi trazido, é permitido à noite imediatamente. E o que é permitido para um judeu mover, mesmo quando o não-judeu trouxe para si mesmo, é apenas dentro de quatro cúbitos ou dentro da cidade se estiver cercada por uma muralha, e deve ser cercada para habitação, ou seja, que foi habitada e depois cercada, e geralmente as cidades são cercadas para habitação, e geralmente as fortalezas não são cercadas para habitação (veja Siman 401).

9. Se houver dúvida se foi trazido de fora do limite, é proibido, e isso se aplica apenas a um não-judeu que não reside na mesma cidade, mas um não-judeu que reside na mesma cidade e frutas que são comuns na cidade, não há preocupação de dúvida. Mesmo se o não-judeu tiver duas casas e uma delas estiver dentro do limite, presume-se que é permitido e é permitido comer, mesmo para quem foi trazido.

10. Um não-judeu que encheu água para seu animal de um poço que é domínio privado para domínio público: é permitido para um judeu dar de beber ao seu animal, desde que o não-judeu não o conheça, para que não haja preocupação de que ele aumente por causa dele. Se ele encheu para o animal de um judeu, é proibido para qualquer uso, mesmo para outro judeu. Se ele encheu de um poço de domínio privado para um carmelita, é permitido para outro, desde que não tenha enchido para ele. Nota: Há quem seja leniente e diga que mesmo se foi trazido através de domínio público para um judeu, é permitido para uma pessoa beber, já que é possível ir lá e beber (Tur em nome de R"T). Há quem permita até mesmo inicialmente (Kol Bo), e assim é o costume de permitir dizer até mesmo inicialmente para um não-judeu trazer cerveja ou outras coisas através de um carmelita ou sem eruv. E embora deva-se ser rigoroso nisso, não se deve impedir os que são lenientes para a necessidade do Shabat e em caso de necessidade, pois há base para ser leniente ao dizer a um não-judeu para uma necessidade, como explicado no Siman 307, e muito mais neste caso.

11. Se um não-judeu colheu ervas para seu animal, se ele não o conhece, um judeu pode alimentar seu animal depois, desde que esteja de frente para ele de forma que não possa desviar-se, pois se for para colocá-lo sobre elas, é proibido, pois há preocupação de que ele possa pegá-las com a mão e comê-las, e elas são muktzeh. Mas se ele o conhece, é proibido, e assim em qualquer caso onde haja preocupação de que ele possa aumentar por causa dele. Mas em um caso onde não há preocupação de que ele possa aumentar por causa dele, como quando ele acendeu uma vela para si mesmo ou fez uma rampa para descer, pois uma vela e uma rampa são suficientes para todos, mesmo que ele o conheça, é permitido.

12. Mesmo que ele não o conheça, se ele disser explicitamente que está fazendo para um judeu, ou mesmo se não disser, mas suas ações indicam que está fazendo para um judeu, como acender uma vela em uma casa onde há um judeu e o não-judeu se foi, é proibido.

13. Se um não-judeu colheu e alimentou o animal de um judeu, não é necessário impedi-lo naquele momento, mas se ele faz isso regularmente, é necessário impedi-lo.

14. Se um não-judeu fez um caixão ou uma sepultura para si mesmo no Shabat, é permitido para um judeu ser enterrado nele. Se ele fez para um judeu, não deve ser enterrado nele para sempre. Isso se aplica apenas se a sepultura estiver em público e o caixão sobre ela, pois todos sabem que foi feito para um certo judeu. Mas se for em segredo, é permitido ser enterrado nele à noite, o tempo suficiente para que pudessem ser feitos. Mesmo quando é em público, é proibido apenas para o judeu para quem foi feito, mas para outro judeu é permitido, desde que espere o tempo suficiente para que pudessem ser feitos.

15. Um não-judeu que trouxe flautas no Shabat para lamentar um judeu: o judeu não deve lamentar com elas, nem ele nem outros, até que espere à noite o tempo suficiente para que pudessem vir de um lugar próximo. Se souber com certeza que foram trazidas de um certo lugar no Shabat, deve esperar à noite o tempo suficiente para que pudessem vir daquele lugar, e então são permitidas tanto para ele quanto para outros. Isso se aplica quando foram trazidas através de domínio público, mas se foram trazidas apenas através de um carmelita, já que não houve transgressão da Torá, não é necessário esperar o tempo suficiente para que pudessem vir, mas são permitidas à noite imediatamente.

16. Se houver dúvida se foram trazidas de fora do limite ou de dentro do limite, há preocupação de que possam ter sido trazidas de fora do limite.


--- EXPLICAÇÃO PRÁTICA ---

O Siman 325 do Shulchan Aruch Orach Chayim aborda as regras sobre o que é permitido ou proibido em relação ao trabalho realizado por não-judeus para judeus no Shabat. Vamos explicar isso de forma prática e acessível:

1. **Convidar Não-Judeus no Shabat**: É permitido convidar um não-judeu para sua casa no Shabat e oferecer comida a ele. Se ele levar a comida para fora, não há problema, desde que ele esteja dentro do pátio. No entanto, se ele estiver fora e pegar algo de dentro, ou se for algo que normalmente é levado para fora, isso é proibido, pois pode parecer que o judeu está pedindo para ele fazer isso.

2. **Caminhos de Paz e Necessidades de Mitzvá**: Em situações onde é importante manter boas relações (caminhos de paz) ou se há um não-judeu agressivo, é permitido dar ou enviar algo através de um não-judeu. Isso também se aplica a necessidades religiosas, como remover chametz (pão fermentado) antes do Pessach.

3. **Troca de Penhores**: Se um não-judeu precisa trocar um penhor que é uma vestimenta, ele pode fazê-lo no Shabat, desde que use a vestimenta. Isso não é considerado comércio. Para um judeu, é melhor que o não-judeu faça a troca sem que o judeu toque no penhor.

4. **Pão Assado por Não-Judeus**: Se um não-judeu assou pão para si mesmo no Shabat, há opiniões divergentes sobre se um judeu pode comê-lo. Em casos de necessidade ou para uma mitzvá, como uma refeição de brit milá, pode-se confiar na opinião que permite.

5. **Peixes e Frutas Colhidos por Não-Judeus**: Se um não-judeu pescou ou colheu frutas para si mesmo, um judeu não pode consumi-los no mesmo dia, mas pode fazê-lo à noite. Se foi feito para um judeu, deve-se esperar o tempo suficiente para que pudessem ser colhidos após o Shabat.

6. **Dúvidas sobre a Intenção do Não-Judeu**: Se houver dúvida sobre se o não-judeu fez algo para um judeu, ou se não se sabe quando foi feito, as regras variam, mas geralmente é necessário esperar até a noite.

7. **Objetos Trazidos de Fora do Limite**: Se um não-judeu trouxe algo de fora do limite permitido no Shabat para si mesmo, um judeu pode usá-lo. Se foi trazido para um judeu, ele pode movê-lo, mas não usá-lo até a noite.

8. **Água para Animais**: Se um não-judeu encheu água para seu próprio animal, um judeu pode usá-la para seu animal, desde que o não-judeu não saiba, para evitar que ele faça mais por causa do judeu.

9. **Ervas para Animais**: Se um não-judeu colheu ervas para seu animal, um judeu pode usá-las, desde que não haja risco de o não-judeu colher mais por causa do judeu.

10. **Trabalho Feito para Judeus**: Se um não-judeu fez algo explicitamente para um judeu, como acender uma vela, é proibido para o judeu usar.

11. **Caixões e Sepulturas**: Se um não-judeu fez um caixão ou sepultura para si mesmo no Shabat, um judeu pode usá-los. Se foi feito para um judeu, não pode ser usado por ele, mas pode ser usado por outro judeu após o Shabat.

12. **Instrumentos Musicais para Lamentação**: Se um não-judeu trouxe instrumentos para lamentação no Shabat, eles não podem ser usados até que se passe o tempo suficiente após o Shabat para que pudessem ter sido trazidos de um local próximo.

Essas regras ajudam a manter a santidade do Shabat, evitando que judeus se beneficiem de trabalhos realizados por não-judeus durante esse dia sagrado.
